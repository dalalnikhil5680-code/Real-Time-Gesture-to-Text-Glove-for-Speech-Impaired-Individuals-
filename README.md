The IoT Glove Project aims to design and develop a smart wearable device
capable of detecting and transmitting hand and finger movements using
embedded sensors and wireless technology. The glove is equipped with flex
sensors to measure the degree of finger bending, which are interfaced with an
ESP32 microcontroller. The microcontroller processes the analog signals from
the sensors, converts them into digital data, and transmits the information
wirelessly through Wi-Fi or Bluetooth to a remote system or application.
The system effectively transforms hand gestures into digital data that can be
used for various applications such as gesture-controlled robotics, prosthetic
hand control, sign language interpretation, and rehabilitation monitoring. This
project demonstrates the practical use of IoT, sensor interfacing, and
embedded systems in developing low-cost, portable, and intelligent human–
